# Button

----- ButtonSize

## size

type: 'small' | 'normal' | 'large'

----- ButtonType

## type

type: 'default' | 'text' | 'primary' | 'success' | 'warning' | 'danger' | 'info'

----- ButtonIcon

## icon

type: string

需要额外引入`font-awesome`，图标名称请参考[font-awesome/icons](http://fontawesome.io/icons/)

----- ButtonDisabled

## disabled

type: boolean

----- ButtonLoading

## loading

type: boolean

----- ButtonFull

## full

type: boolean

----- ButtonNativeType

## nativeType

type: 'button' | 'submit' | 'reset'

----- ButtonOnClick

## onClick

type: (e) => Promise | void
