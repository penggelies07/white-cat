# Notification 提示条

----- NotificationType

## `type`?: 'primary' | 'success' | 'warning' | 'error'

名称

----- NotificationIcon

## `icon`?: string

图标

----- NotificationContent

## `content`?: string

内容

----- NotificationClosable

## `closable`?: boolean = false

是否可点击关闭

----- NotificationDuration

## `duration`?: number

显示持续时间
