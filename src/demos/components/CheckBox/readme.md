# CheckBox 多选框

----- CheckBoxChecked

## `checked`?: boolean = false

是否选中

----- CheckBoxValue

## `value`?: any

选中值

----- CheckBoxDisabled

## `disabled`?: boolean = false

是否禁用

----- CheckBoxSize

## `size`?: 'small' | 'normal' | 'large' = 'normal'

尺寸

----- CheckBoxOnChange

## `onChange`?: (checked: boolean, value?: any) => void

`checked`发生改变时的回调函数
