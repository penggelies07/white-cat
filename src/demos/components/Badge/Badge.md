# Badge 徽标数

----- BadgeValue

## `value`?: number | string = 0

当前值

----- BadgeMax

## `max`?: number = 99

最大值，仅`value`为`number`时有效

----- BadgeType

## `type`?: 'primary' | 'success' | 'warning' | 'danger' | 'gray' = 'danger'

样式类型

----- BadgeDot

## `dot`?: boolean = false

圆点形式

----- BadgeChildren

## `children`?: React.ReactNode

浮动目标

----- BadgeOffset

## `offset`?: {top: string, right: string}

浮动偏移
